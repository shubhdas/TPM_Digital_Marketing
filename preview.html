Digital Marketing Workspace - Component Preview

This page demonstrates the various LWC components that make up the Digital Marketing Workspace.

App Workspace (Main Container)

The main workspace that contains all the functional tabs:

  • Home Dashboard
  • Campaigns Manager
  • Clients Manager
  • Leads Manager
  • Projects Manager
  • Billing Manager
  • Google Ads Integration

This component serves as the main container for the Digital Marketing application.

Admin Dashboard

Provides an overview of key metrics and analytics for digital marketing activities.

Campaigns Manager

Manage marketing campaigns including budget, dates, and status tracking.

Clients Manager

Manage client information and relationships.

Leads Manager

Track and manage marketing leads.

Projects Manager

Manage digital marketing projects and their timelines.

Billing Manager

Handle billing and financial aspects of marketing activities.

Google Ads Integration

Integration with Google Ads for campaign management.
